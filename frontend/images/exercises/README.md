# Exercise Images

This folder contains images for exercises displayed on the exercises page.

## Required Images

Place the following image files in this directory:

### Arms
- `bicep-curl.jpg` - Bicep Curl exercise
- `tricep-dip.jpg` - Tricep Dip exercise

### Legs
- `squat.jpg` - Barbell Squat exercise
- `lunges.jpg` - Walking Lunges exercise

### Chest
- `bench-press.jpg` - Bench Press exercise
- `push-ups.jpg` - Push-Ups exercise

### Abs
- `crunches.jpg` - Crunches exercise
- `plank.jpg` - Plank exercise

### Back
- `pull-ups.jpg` - Pull-Ups exercise
- `bent-over-row.jpg` - Bent-Over Row exercise

## Image Specifications

- **Format**: JPG or PNG
- **Recommended size**: 400x300 pixels or larger
- **Aspect ratio**: 4:3 or 16:9
- **File size**: Keep under 500KB for optimal loading

## Fallback

If images are not found, the page will automatically display placeholder SVGs with the exercise name.
